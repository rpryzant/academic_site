Matrix Identity Cheat Sheet

Let $A$ be a matrix. Then following matrices have fancy names:

  • Transpose: $A^T$, where $A = A^T$

  • Conjugate transpose: $A^*$ or $A^H$. When $A$ is real, we have $A^T = A^H$. Otherwise, we have $\overline{A^T} = A^H$.

  • Symmetric: when $A = A^T$.

  • Hermitian: when $A^T = A^H$.

  • Skew-Symmetric: when $A^T = -A$.

  • Skew-Hermitian: when $A^T = -A^H$.

  • Orthogonal: when $A^T = A^{-1}$. This implies orthonormal columns.

  • Unitary: when $A^H = A^{-1}$

  • Normal: when $A^H A = A A^H$
